Help & Tutorial
Getting started ›

Welcome to the Peter Hoff Design CMS. This panel lets you manage all portfolio projects visible on the public website.

Use the toolbar at the top to filter projects by category, add new projects, edit the site bio, or come back here for help.

All changes are saved to the database immediately — there is no draft mode. Toggle a project's Visible switch to hide it from the live site without deleting it.

Adding & editing projects ›

Add Project — Click the green button in the toolbar. Fill in the project name and choose a category (required). The slug is auto-generated from the name but can be edited manually.

Edit — Click the pencil icon on any row to open the edit modal. Switch between the Details and Images tabs.

Delete — Click the trash icon. This permanently removes the project and all its uploaded images. You will be asked to confirm.

Reorder — Drag rows up or down using the ⠿ handle on the left. The order here matches the order on the live site.

Uploading & managing images ›

Open a project → switch to the Images tab. Click the upload zone or drag files onto it. Supported formats: JPG, PNG, WebP, GIF, MP4.

Files up to 100 MB each. Multiple files can be selected at once.

Reorder images — Drag thumbnails left/right using the ⠿ handle at the bottom of each thumbnail. The first image is used as the project cover on the grid.

Photographer credit — Type a name in the credit field below each image. It appears as a caption on the detail page and in the lightbox.

Delete image — Hover the thumbnail and click the ✕ button. The file is permanently deleted from storage.

Categories ›

Categories are predefined and managed in the database. Each project belongs to exactly one category.

The filter tabs on the toolbar and on the public site are built automatically from whichever categories have at least one visible project. Empty categories are hidden.

To add or rename a category, contact your developer — it requires a database change.

Location map ›

In the Details tab, the Plats / Adress field accepts any address or place name (e.g. "Stortorget 1, Malmö").

A live map preview appears below the field as you type. The same map is shown at the bottom of the project page on the public site.

Leave the field blank to hide the map section entirely.

Bio / Intro text ›

Click Bio / Intro in the toolbar to open the text editor. This text appears in the footer of the public website.

Use it for a short studio description, credentials, or any introductory copy. Line breaks are preserved.

Click Spara to save. The change is live immediately.

Visibility & the live site ›

The Visible toggle in the project row (and in the Details tab) controls whether the project appears on the public site.

Hidden projects still exist in the database and can be made visible again at any time.

The View Site ↗ link in the header opens the live public website in a new tab so you can check your changes.

The public site requires the password phd123! to enter during the under-construction phase.
